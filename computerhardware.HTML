Computer Hardware

What are the parts of a computer

Motherboard

A motherboard is one of the main components for a computer that is needed in order for the computer to run and it allows all of the other components in a computer be able to communicate with each other

CPU

The CPU is the part of the computer that is kind of like the brain of the computer and it controls all of the other parts and this part is also very necessary for modern day computers

RAM

A RAM allows the computer to do more than one thing at a time without something happening to the computer which is also very necessary since everyone is using computers now in days

Hard Drive

A Hard Drive is necessary to store lots of data in the computer and this is very helpful so that way you can save something instead of just having to do it all over again

Motherboard

All the parts communicate with each other by sending each other signals so that they can all communicate and all the signals go through the motherboard

The Basic's

The keyboard is one of the three base items for a computer and this allows you to type on the screen, the mouse helps you so that you can click on different things so that you can type on it, the monitor is used so that you can see what you are doing
